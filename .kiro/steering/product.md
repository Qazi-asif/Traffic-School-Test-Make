# Product Overview

Multi-state online traffic school and defensive driving course platform. Provides state-compliant driver education courses with certificate generation and submission to state authorities.

## Core Features

- Multi-state support (Florida, Missouri, Texas, Delaware)
- Online course delivery with chapter-based learning
- Quiz and final exam system with passing requirements
- Certificate generation and state submission (FLHSMV DICDS integration)
- Payment processing (Stripe, PayPal)
- User registration with multi-step validation
- Course timer enforcement for compliance
- Email notifications and receipts
- Admin dashboard for course and user management

## User Roles

- Student: Takes courses, receives certificates
- Instructor: Manages course content
- School Admin: Manages school operations
- Super Admin: Full system access

## State-Specific Features

Each state has unique compliance requirements:
- Florida: DICDS integration, FLHSMV SOAP submissions
- Missouri: Form 4444 generation, specific quiz banks
- Texas/Delaware: State-specific course structures and certificates
